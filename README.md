# 💰 Finance Dashboard UI

A scalable and user-friendly financial dashboard for tracking and analyzing personal finances. Built with modern frontend technologies to deliver an interactive and responsive user experience.

---

## 🚀 Live Demo

👉 https://finance-dashboard-ui-sigma-ten.vercel.app

---

## 🔗 Repository

👉 https://github.com/anushravya/FinanceDashboardUI

---

## 📌 Features

* 📊 Dashboard overview with summary cards (Balance, Income, Expenses)
* 📈 Time-based visualization (Balance trend over months)
* 🥧 Categorical visualization (Spending breakdown)
* 📋 Transaction list with details
* 🔍 Transaction filtering, sorting, and search
* 🧠 Insights section for better financial understanding
* 🎭 Role-based UI (Viewer/Admin)
* 🌗 Dark mode support
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **Frontend:** React.js (TypeScript)
* **Styling:** Tailwind CSS
* **State Management:** Context API
* **Build Tool:** Vite
* **Deployment:** Vercel

## ⚙️ Installation & Setup

1. Clone the repository:

```id="r2k7os"
git clone https://github.com/anushravya/FinanceDashboardUI.git
```

2. Navigate to project folder:

```id="z7x1ac"
cd FinanceDashboardUI
```

3. Install dependencies:

```id="n8q1hy"
npm install
```

4. Run the project:

```id="k3v0lp"
npm run dev
```
